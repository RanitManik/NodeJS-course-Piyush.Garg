REST API - JASON

GET/users => render an HTML Document
GET/api/users => list all users in JSON

GET/users/1 => Get the user with the ID 1
GET/api/users/2 => Get the user with the ID 2

// Dynamic Path Parameters => GET/api/users/:id
:id -> variable | Dynamic

POST/users => render an HTML Document
POST/api/users => create a new user

PATCH/user/1 => render an HTML Document
PATCH/api/user/1 => edit the user with the ID 1

DELETE/user/1 => render an HTML Document
DELETE/api/user/1 => delete the user with the ID 1
